# Injection Toolbox (注入工具箱)

Injection Toolbox 是一款功能强大的 Android APK 极速无损注入与配置工具。它专为开发者和高级用户设计，旨在通过自动化手段简化 XP 模块配置以及组件工厂（ComponentFactory）的注入流程。

## 🚀 核心功能

### 1. XP 模块动态配置 (XP Config)
- **JSON 实时解析与修改**：直接读取并修改 APK 内 `assets/kong.json` 配置文件。
- **防检测混淆**：
    - **SO 库物理改名**：随机生成 SO 文件名，有效规避简单的文件名检测。
    - **配置重写**：自动化处理 JSON 转义符，确保配置文件的兼容性。
- **自定义 UI 设置**：支持配置开发者 ID、渐变水印颜色、背景音乐外链等。

### 2. 组件工厂注入 (Component Factory Injection)
- **深度注入与合并**：利用 Component Factory 技术将模块无损注入到目标 APK 中。
- **过 VM 壳技术**：采用高级注入方案，尝试绕过虚拟机壳检测。
- **极速处理**：优化的注入逻辑，在手机端即可完成复杂的合并操作。

### 3. 现代化 UI 设计
- **Jetpack Compose 驱动**：基于最新的原生 UI 框架构建，响应迅速。
- **分类配置卡片**：将基础配置、UI 视觉、核心防封设置分类展示，清晰易读。
- **实时日志反馈**：内置日志显示区域，实时追踪处理进度与错误。

## 🛠 使用说明

1.  **授权**：首次启动请授予“所有文件访问权限”，以便工具读写存储空间中的 APK 文件。
2.  **选择文件**：
    - 在“XP 模块配置”页选择需要修改配置的模块包。
    - 在“组件工厂注入”页分别选择模块包和目标应用包。
3.  **参数配置**：根据需求填写开发者 ID、颜色、杀 EN 规则等。
4.  **执行任务**：点击“生成”或“执行注入”按钮。
5.  **输出路径**：处理后的 APK 默认保存至手机根目录下的 `Download/` 文件夹中。

> **注意**：生成的 APK 可能需要使用 MT 管理器或其他工具进行 V2+Zipalign 签名后才能正常安装。
